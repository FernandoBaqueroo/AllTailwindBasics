Diseño Web con Tailwind CSS

Sección Lobster

Este párrafo utiliza la fuente Lobster, una fuente cursiva que da un toque creativo al texto. Puedes personalizar aún más las fuentes importando diferentes estilos desde Google Fonts.

Sección Roboto

Este párrafo usa la fuente Roboto, una de las fuentes sans-serif más populares para el diseño web moderno. Es clara y legible en varios tamaños de pantalla.

Sección Responsiva con Monospace

Este es un ejemplo de cómo las fuentes Monospace pueden ser utilizadas para mostrar código o texto que necesita alinearse uniformemente. Además, el diseño es completamente responsivo. Redimensiona la ventana para ver cómo se ajusta el contenido.
